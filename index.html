Features

Browse resources

Navigate workloads, services, config, storage, and custom resources.

Inspect details

Read status, YAML, properties, and events in one desktop workspace.

Work with pods

Follow logs and inspect pod state without rebuilding context.

Edit YAML

Review and apply Kubernetes manifests from a focused editor workflow.

Visualize relationships

See owner references and related resources without manually tracing labels.

Multi-cluster workflow

Connect to saved clusters and keep navigation organized per workspace.

Resource events

Read recent events next to the resource you are already inspecting.

Custom resources

Browse CRDs alongside built-in Kubernetes resources.

Desktop app

Use a native interface on Windows, macOS, and Linux.
